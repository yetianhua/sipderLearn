## 第三方库

- 需要使用到的库已经放在requirements.txt，使用pip安装的可以使用指令  
  `pip install -r requirements.txt`
- 如果国内安装第三方库比较慢，可以使用以下指令进行清华源加速
  `pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/`
